The packages ship `.d.ts` for every public export. Two snippets cover most use cases — paste them into your project as a reminder of the public shape.

## Document

```ts
interface Document {
    id: string;
    name: string;
    pageSize: PageSize;
    orientation: PageOrientation;
    pages: Page[];
    sections: Section[];
    createdAt: number;
    updatedAt: number;
}
```

## Page

```ts
interface Page {
    id: string;
    name?: string;
    background?: {
        color?: string;
        imageUrl?: string;
    };
    guides?: PageGuides;
}

interface PageGuides {
    vertical: number[];
    horizontal: number[];
}
```

## Section

```ts
type Section = ImageSection | TextSection | ShapeSection | DrawingSection | FrameSection;

interface BaseSection {
    id: string;
    pageIndex: number;
    x: number;
    y: number;
    width: number;
    height: number;
    zIndex?: number;
    hidden?: boolean;
    locked?: boolean;
    parentFrameId?: string; // set when dropped inside a container FrameSection
}

interface ImageSection extends BaseSection {
    type: 'image';
    imageUrl?: string;
    fit?: 'cover' | 'contain' | 'stretch' | 'crop';
    crop?: { x: number; y: number; width: number; height: number };
    maskShape?: 'rect' | 'circle' | 'line'; // placeholder frame — clips the image to a shape
}

interface TextSection extends BaseSection {
    type: 'text';
    content: string;
    fontSize: number;
    fontFamily: string;
    color: string;
    align: 'left' | 'center' | 'right';
}

interface ShapeSection extends BaseSection {
    type: 'shape';
    shape: 'rect' | 'ellipse' | 'line';
    fill?: string;
    stroke?: string;
    strokeWidth?: number;
}

interface DrawingSection extends BaseSection {
    type: 'drawing';
    strokes: Array<{ color: string; width: number; points: Array<{ x: number; y: number }> }>;
}

interface FrameSection extends BaseSection {
    type: 'frame';
    fill?: string; // 'transparent' (default) draws a pure grouping box
    stroke?: string; // 'transparent' (default) → no border
    strokeWidth?: number; // border width in points; default 1
    radius?: number; // corner radius in points; default 0
}
```

A **container frame** (`FrameSection`) is a box that owns other sections: any
section dropped inside it carries that frame's id in `parentFrameId`, and the
frame draws behind its children. A **placeholder frame** is an `ImageSection`
with `maskShape` set — a shaped image slot. See the
[Frames concept](/docs/concepts/frames) for the full model.

## Page sizes

```ts
type PageSize =
    | 'A0'
    | 'A1'
    | 'A2'
    | 'A3'
    | 'A4'
    | 'A5'
    | 'LETTER'
    | 'LEGAL'
    | 'TABLOID'
    | 'EXECUTIVE'
    | 'STATEMENT'
    | string; // also accepts custom sizes registered via createPageSize

type PageOrientation = 'portrait' | 'landscape';
type MeasurementUnit = 'mm' | 'in' | 'px';
```
